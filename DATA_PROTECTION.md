# 🛡️ הגנה על הנתונים - מדריך מלא

## ⚠️ מצב נוכחי:

כרגע הנתונים שלך נשמרים ב-**localStorage** של הדפדפן.

### 🔴 הבעיה:
- **localStorage נמחק כשמנקים את הדפדפן**
- **עדכון גרסה יכול למחוק נתונים**
- **אין גיבוי אוטומטי**

### ✅ הפתרון:
**מעבר ל-Supabase** - בסיס נתונים בענן (כבר מוכן במערכת!)

---

## 🚀 איך לעבור ל-Supabase (3 שלבים):

### שלב 1: הגדרת Supabase (10 דקות)

1. **צור חשבון ב-Supabase** (בחינם):
   - גש ל-https://supabase.com
   - לחץ "Start your project"
   - צור פרויקט חדש בשם "clickef-crm"

2. **קבל את המפתחות**:
   - בפרויקט שלך, לחץ על ⚙️ Settings
   - לחץ על "API"
   - העתק את:
     - `Project URL` (מתחיל ב-https://)
     - `anon public key` (מתחיל ב-eyJ...)

3. **הזן את המפתחות במערכת**:
   - פתח את הקובץ `.env.local`
   - עדכן:
     ```
     VITE_SUPABASE_URL=https://YOUR_PROJECT.supabase.co
     VITE_SUPABASE_ANON_KEY=eyJ...
     ```

### שלב 2: יצירת הטבלאות (5 דקות)

1. ב-Supabase, לחץ על "SQL Editor"
2. פתח את הקובץ `CREATE_TABLES.sql` שנמצא בפרויקט
3. העתק את כל התוכן
4. הדבק ב-SQL Editor ולחץ "Run"
5. הטבלאות ייווצרו אוטומטית! ✅

### שלב 3: העברת הנתונים (דקה אחת!)

1. פתח את המערכת בדפדפן
2. פתח את ה-Console (F12)
3. הקלד:
   ```javascript
   await window.migrateFromLocalStorage()
   ```
4. לחץ Enter
5. **כל הנתונים שלך יועברו ל-Supabase!** 🎉

## 🔄 גיבוי ידני (עד שתעבור ל-Supabase):

### אפשרות 1: ייצוא ל-CSV
1. בלוח אירועים/לקוחות/לידים
2. לחץ על כפתור "ייצוא"
3. שמור את הקובץ במחשב

### אפשרות 2: גיבוי localStorage
1. פתח Console (F12)
2. הקלד:
   ```javascript
   const backup = localStorage.getItem('ME_CFM_STORAGE_V12');
   console.log(backup);
   ```
3. העתק את כל הטקסט ושמור בקובץ txt

### אפשרות 3: שכפול הדפדפן
- אל תנקה את הדפדפן!
- אל תמחק cookies/cache אלא אם כן גיבית

## 🆘 מה לעשות אם הנתונים נמחקו:

### אם יש לך גיבוי CSV:
1. לחץ על "ייבוא" בלוח הרלוונטי
2. בחר את קובץ ה-CSV
3. הנתונים יחזרו!

### אם יש לך גיבוי localStorage:
1. פתח Console (F12)
2. הקלד:
   ```javascript
   localStorage.setItem('ME_CFM_STORAGE_V12', 'הדבק כאן את הגיבוי');
   location.reload();
   ```

### אם עברת ל-Supabase:
- **אין בעיה!** הנתונים בענן ולא יימחקו 🎉
