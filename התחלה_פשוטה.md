# 🚀 העלאת המערכת לאינטרנט – הדרכה פשוטה

**מטרה:** שהמערכת תהיה זמינה באינטרנט כל הזמן, עם קישור אחד שתשתמש בו.

---

## לפני שמתחילים

- צריך **חיבור אינטרנט**.
- תצטרך **להרשם ב-Vercel** (חינם) – עם אימייל או עם Google.

---

## שלב 1: לפתוח את "הטרמינל"

1. לחץ על **Windows** (כפתור Start).
2. הקלד: **PowerShell**
3. כש־"Windows PowerShell" מופיע – לחץ עליו (או Enter).

**תיפתח חלונית שחורה עם טקסט לבן – זו הטרמינל.**

---

## שלב 2: להתקין את הכלי של Vercel (פעם אחת בלבד)

1. **העתק** את השורה הזו (בדיוק כמו שכתוב):
   ```
   npm install -g vercel
   ```
2. **לחץ ימני** בתוך החלונית השחורה (הטרמינל) – זה ידביק את השורה.
3. לחץ **Enter**.
4. חכה עד שהשורות מפסיקות לרוץ (יכול לקחת דקה־שתיים).
5. אם בסוף אין כתוב "error" באדום – **ההתקנה הצליחה.**

---

## שלב 3: להיכנס לתיקיית הפרויקט

1. **העתק** את השורה הזו:
   ```
   cd "c:\Users\USER\Desktop\my_app"
   ```
2. **לחץ ימני** בטרמינל (כדי להדביק).
3. לחץ **Enter**.

אחרי זה אתה "נמצא" בתיקייה של המערכת.

---

## שלב 4: להעלות ל-Vercel

1. **העתק** את המילה:
   ```
   vercel
   ```
2. **הדבק** בטרמינל (לחץ ימני).
3. לחץ **Enter**.

---

## שלב 5: מה שיקרה עכשיו

1. **בפעם הראשונה** – יכול להיפתח לך **דפדפן** (Chrome / Edge וכו').
   - אם שואלים "Log in" או "Sign up" – **הירשם** (עם אימייל או Google).
   - אחרי ההתחברות אפשר **לסגור את הדפדפן** ולחזור לחלונית השחורה.

2. **בטרמינל** יופיעו שאלות. בכל פעם שמופיעה שאלה:
   - אם כתוב **Y** או **N** – לרוב מספיק ללחוץ **Enter** (מקבלים את ההמלצה).
   - אם שואלים "Project name?" – פשוט **Enter** (או תכתוב שם כמו: kalikif).

3. **בסוף** יופיע שורה עם **קישור**, למשל:
   ```
   https://my-app-xxxx.vercel.app
   ```
   **זה הקישור למערכת שלך באינטרנט.**

4. **העתק את הקישור** ושמור אותו (במסמך / במייל לעצמך).  
   מעכשיו אפשר לפתוח את המערכת מהדפדפן עם הקישור הזה – **בלי להריץ שום פקודה**.

---

## אם משהו לא עובד

- **"self-signed certificate" או SELF_SIGNED_CERT_IN_CHAIN (שגיאה באדום)**  
  → זו בעיית תעודת אבטחה ברשת (למשל באינטרנט רימון). **עשה את זה:**
  1. בטרמינל, **הדבק** את השורה הזו ולחץ Enter:
     ```
     npm config set strict-ssl false
     ```
  2. אחר כך **שוב** הרץ את הפקודה משלב 2:
     ```
     npm install -g vercel
     ```
  אם עדיין לא עובד – נסה מהרשת הביתית או מרשת סלולר (נקודה חמה).

- **שגיאה ב-vercel: "request to https://vercel.com/... failed, reason: self-signed certificate"**  
  → הרשת חוסמת גם את החיבור ל-Vercel. **נסה כך (שורה אחת, ואז Enter):**
  1. וודא שאתה בתיקייה: `cd "c:\Users\USER\Desktop\my_app"` ולחץ Enter.
  2. **הדבק את השורה הזו בשלמות** ולחץ Enter **פעם אחת**:
     ```
     $env:NODE_TLS_REJECT_UNAUTHORIZED="0"; vercel
     ```
  אם עדיין לא עובד – נסה מהרשת הביתית או מטלפון (נקודה חמה), או העלאה ידנית ב־Vercel (ראה "אופציה: העלאה מהדפדפן" בהמשך).

- **"לא מזהה את הפקודה npm"**  
  → צריך Node.js במחשב. אפשר להוריד מכאן: https://nodejs.org (גרסה LTS) ולהתקין, ואז לחזור לשלב 2.

- **נפתח דפדפן ולא קורה כלום**  
  → התחבר/הירשם ב-Vercel, אחר כך חזור לטרמינל ולחץ Enter.

- **כתוב error באדום (אחר)**  
  → צלם מסך של החלונית השחורה (כל הטקסט) ושלח למישהו שמבין, או העתק את הטקסט ושלח.

## סיכום

| מה עשית | למה |
|--------|-----|
| פתחת PowerShell | כדי להריץ פקודות |
| הרצת `npm install -g vercel` | התקנת כלי להעלאה (פעם אחת) |
| הרצת `cd "c:\Users\USER\Desktop\my_app"` | כניסה לתיקיית הפרויקט |
| הרצת `vercel` | העלאת האתר ל-Vercel וקבלת קישור |
