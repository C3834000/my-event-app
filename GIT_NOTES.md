# 📝 Git Notes - תזכורת

## פקודות בסיסיות:

1. `git status` - בדוק מה השתנה
2. `git add .` - סמן לשמירה
3. `git commit -m "תיאור"` - שמור
4. `git log --oneline` - צפה בהיסטוריה
5. `git push origin main` - העלה לGitHub

## כללי אצבע:
- שמור לאחר כל תיקון משמעותי
- תן תיאורים ברורים
- אל תפחד לנסות - אפשר תמיד לחזור אחורה!
